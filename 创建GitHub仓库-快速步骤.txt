📋 在 GitHub 上创建仓库 - 快速步骤

1. 打开浏览器，访问：https://github.com/new

2. 登录你的 GitHub 账号（如果还没登录）

3. 填写仓库信息：
   - Repository name（仓库名称）：输入 mathmaster
   - Description（描述）：可以留空，或输入 "高中数学学习平台"
   - 选择 Public（公开）
   - ⚠️ 重要：不要勾选 "Add a README file"
   - ⚠️ 重要：不要勾选 "Add .gitignore"
   - ⚠️ 重要：不要勾选 "Choose a license"

4. 点击绿色的 "Create repository" 按钮

5. 创建成功后，你会看到一个页面，显示 "Quick setup" 或 "…or push an existing repository from the command line"

6. 复制这个页面上显示的仓库地址（类似：https://github.com/你的用户名/mathmaster.git）

7. 告诉我你的 GitHub 用户名，我会帮你完成剩下的步骤！
